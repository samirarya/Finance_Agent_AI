💰 Project Sammy: Personal Finance Agent

This report summarizes the complete development cycle and architecture of Sammy, a state-of-the-art multi-agent conversational AI system designed to democratize financial literacy.

1. Multi-Agent Orchestration (CrewAI)

Hierarchical Architecture

Implemented a sophisticated Hierarchical (Manager-Led) architecture comprising 6 specialized agents and 1 Workflow Router:

  • Workflow Router (Manager): Dynamically interprets queries and delegates to subordinates.
  • Finance Q&A Agent: Focused on general education and terminology.
  • Portfolio Analysis Agent: Quantitative analyst for risk and diversification.
  • Market Analysis Agent: Strategist for real-time stock insights.
  • Goal Planning Agent: Certified planner for retirement and savings projections.
  • News Synthesizer Agent: Journalist for market context and sentiment.
  • Tax Education Agent: Specialist for tax-advantaged accounts and implications.

2. Knowledge Base & RAG Implementation

Pinecone Vector Store

  • Curated Content: Generated and indexed 110 financial education articles covering Stocks, Bonds, ETFs, and specialized Options Trading.
  • Semantic Search: Integrated Pinecone with high-dimensional (3072-dim) embeddings using models/gemini-embedding-001.
  • Robust Indexing: Developed a rate-limit optimized indexer to safely populate the cloud knowledge base.

3. Three-Tier LLM Resilience

Automated Fallback Sequence

To ensure 100% availability even during cloud quota peaks, Sammy features an automated fallback logic:

  • Tier 1: Gemini 2.5 Flash (Primary Cloud - High performance)
  • Tier 2: OpenAI GPT-4o (Secondary Cloud - High reliability)
  • Tier 3: Local Llama 3.1 8B (Ollama) (Tertiary - Local hardware fallback)

Refined the local model workflow to a Sequential Process to ensure accurate tool calling without reasoning overhead.

4. Specialized Tooling & Accuracy

Hallucination-Proof Facts

  • Market Data: Custom tool utilizing the Alpha Vantage API for real-time global quotes.
  • Portfolio Management: Developed a meticulous CSV tool that reads sample_portfolio.csv using absolute pathing and KWarg filtering to prevent model hallucinations about data locations.
  • Calculators: Custom Python logic for compound interest and savings goals.

5. Observability & Monitoring

LangSmith Integration

Connected the entire agent loop to LangSmith, providing full transparency into:

  • Agent decision-making and delegation trees.
  • Individual tool execution latency and outcomes.
  • Token usage and prompt performance.

6. Deployment & Cloud Sync

Streamlit Community Cloud

  • Flattened Structure: Optimized the GitHub repository for cloud hosting.
  • Python 3.12 Compatibility: Forced stable versioning via .python-version to resolve Pydantic V1 conflicts.
  • Automated CI/CD: Fully synchronized with https://github.com/samirarya/Finance_Agent_AI.git for instant updates.
